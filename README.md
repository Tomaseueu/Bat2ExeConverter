# Bat2ExeConverter

## This is custom Bat2Exe Converter made in C++ programming language

## What can you do with our Bat2Exe
- Convert BAT to EXE

# How to use:
- Download .exe Release or Convert C++ File to exe
- Go to CMD write file name for example bat2exe.exe then your bat file so bat2exe.exe test.bat and then how you want to name the final exe (warning required to add .exe to the end) bat2exe.exe test.bat test.exe
